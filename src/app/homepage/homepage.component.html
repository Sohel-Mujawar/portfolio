Hangman Game

This project is a classic word-guessing game called Hangman. The player tries to guess the hidden word by suggesting letters within a certain number of guesses. Each incorrect guess brings the player closer to losing by displaying parts of a hangman figure. The game is won when the player correctly guesses all the letters of the word before the hangman figure is fully drawn.
