Simple Calculator with Dark and Light Mode

Project Description

This project is a basic calculator that allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division. It includes a dark and light mode toggle feature, enabling users to switch between the two themes for a more personalized experience.

Technology Used

  • HTML: Defines the structure and layout of the calculator.
  • CSS: Provides the styling for the calculator, including dark and light themes.
  • JavaScript: Handles the calculator's functionality and theme switching logic.

Features

  • Basic Arithmetic Operations: Supports addition, subtraction, multiplication, and division.
  • Theme Toggle (Dark/Light Mode): A button to switch between dark mode and light mode.
  • Responsive Design: The layout adjusts to different screen sizes, making it accessible on both desktop and mobile devices.
  • User-Friendly UI: Simple and clean interface for ease of use.

Hangman Game Project

Project Description

This project is a classic word-guessing game called Hangman. The player tries to guess the hidden word by suggesting letters within a certain number of guesses. Each incorrect guess brings the player closer to losing by drawing parts of a hangman figure.

Technology Used

  • HTML: Structure of the game (gameboard, input fields, etc.)
  • CSS: Styles the game elements, gives visual feedback, and displays the hangman figure.
  • JavaScript: Implements game logic, checks guesses, handles resets.

Features

  • Random word selection from a predefined list.
  • Feedback on correct and incorrect guesses.
  • Win/Loss conditions based on guessing the word or completing the hangman figure.
  • Responsive design for different screen sizes.
